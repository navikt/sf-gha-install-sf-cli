# Install SF CLI

Install SF CLI with `npm`

## Usage

<!-- Start usage -->
```yaml
- uses: navikt/sf-gha-install-sf-cli@<tag/sha>
    with:
        # Version for the SF CLI version
        # Required: false
        # Default: '2.104.6'
        version: ''
```
<!-- end usage -->
